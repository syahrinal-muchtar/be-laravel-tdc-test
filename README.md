## How to run the app

1. Create database using MySQL with name `tdc_db`
2. run command `php artisan migrate:fresh --seed` in terminal
3. run command `composer install`
4. run `php artisan serve`
5. you can access the API from url `http://localhost:8000/api`

Import API Collection from https://github.com/syahrinal-muchtar/be-laravel-tdc-test/blob/master/Test-TDC.postman_collection.json
